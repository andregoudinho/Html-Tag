Titulo h6

Os titulos são feitos usando a tag "h", sendo usado h1 ate o h6, quanto maior o numero menor o tamanho da fonte

Para escrever textos é utilizado a tag "p"


A tag "br" é usada para quebrar linhas (pular linahs)

A tag "hr é usada para criar uma linha"

A tag "a" é usada para colocar um link no site

A tag "span" é usada para poder dar destaque a uma frase dentro de um paragrafo.

Quando queremos usar uma imagem, usamos a tag "img"

A tag "div" é usada para criar uma divisória dentro do código.

Dentro da tag "body" é onde o codigo deve ser escrito.

A tag "style" é usada para implementar o css dentro do código.
